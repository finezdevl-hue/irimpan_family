Automatic Daily Reminders (Cron)

To send reminders automatically every morning, add this line to your server's crontab (crontab -e): 

# Send birthday/death anniversary WhatsApp reminders daily at 8:00 AM
0 8 * * * cd /path/to/family_tree && /path/to/venv/bin/python manage.py send_birthday_reminders >> /var/log/birthday_reminders.log 2>&1

For a dry-run (no messages sent, just preview): add --dry-run flag.

On Render.com: use a Cron Job service pointing to
python manage.py send_birthday_reminders with schedule 0 8 * * *
